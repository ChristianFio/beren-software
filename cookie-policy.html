Cookie Policy

Last updated: April 25, 2026

The website berensofware.com uses only essential technical cookies, for which prior user consent is not required, pursuant to art. 122 of the Italian Privacy Code and the Garante guidelines of June 10, 2021.

No profiling cookies. No third-party analytics. No data shared with advertising platforms.

What cookies are

Cookies are small text files that visited sites send to the user's browser, where they are stored to be retransmitted to the site on the next visit. Cookies can be "technical" (necessary for site operation) or "profiling" (aimed at building user profiles).

Beren Software stores these preferences via the browser's localStorage, which performs the same function as a technical cookie but remains exclusively on the user's device and is not transmitted to the server.

Third-party resources

The Site loads resources from external providers exclusively for technical and rendering purposes. None of them install profiling cookies:

  • Bunny Fonts (fonts.bunny.net): loads the Inter and JetBrains Mono fonts. GDPR-native service hosted in the European Union, explicitly designed not to log end-user IP addresses and not to use cookies. Bunny.net Privacy Policy.
  • Web3Forms (api.web3forms.com): processes the contact form. The connection happens only when the user submits the form, never automatically on page load. No persistent cookies. Web3Forms Privacy Policy.
  • Cal.com (cal.com): opens the booking page. Connection only happens if the user clicks the link. Cal.com Privacy Policy.

Regulatory note: we deliberately replaced Google Fonts with Bunny Fonts to avoid transferring IP addresses to US servers (Google LLC), in line with consolidated practice from the Italian Garante (see decision of July 11, 2022 on extra-EU transfers).

Information banner

On first visit the Site shows an informational banner that, in compliance with art. 122 of the Privacy Code and the Garante guidelines of June 10, 2021, has purely informational function and does not represent a consent collection mechanism (not required for technical cookies). The banner can be dismissed with a single click on the "Got it" button. It can always be reopened from the "Cookie preferences" link in the footer.
